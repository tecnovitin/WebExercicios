Anotações
<nav> - Indica a area de navecação do site
<ul> e <li> Conectam os "botões" para um outro link

<header class="intro"> Define a area, tipo um body
<div class="overlay"></div> Pocisiona a imagem para o fundo do site e deixa ela "transparente"; Melhora as letras.
<div class="conteudo"> Para colocar texto. (<h1> e outros seram colocados dentro do <div>)

<section class="sobre"> Classe sobre

No CSS

* { ... } - Seleciona todos os elementos do site
margin: 0 - Remove as margens e preenchimentos

body { ... } - Estiliza o corpo da página.
font-family - Arial, sans-serif;: Define a fonte utilizada.
color: #333 - Define a cor padrão do texto.
line-height: 1.6 - Define o espaçamento entre linhas, melhorando a legibilidade.

nav { ... } - Estiliza a barra de navegação.
background: #333 - Define uma cor de fundo escura para o menu.
padding: 10px 0 - Adiciona espaçamento interno na parte superior e inferior.

nav ul { ... } - Remove as marcações padrão da lista (list-style: none)  
utiliza flexbox (display: flex) para alinhar os itens horizontalmente.
justify-content: center - Centraliza os itens do menu.

nav ul li { ... } - Espaçamento entre os botões/itens

nav ul li a { ... } - Estiliza os links do menu.
color: #fff - Define a cor branca para os textos dos links.
text-decoration: none - Remove o sublinhado dos links.
font-weight: bold - Deixa o texto em negrito.


header.intro { ... } - Aplica estilos à seção de introdução (header com classe intro).
position: relative - Define que os elementos posicionados internamente podem se posicionar relativamente a este container.
height: 100vh - Define a altura como 100% da viewport (tela), fazendo com que a seção ocupe toda a altura da janela.
background-image: url('imagens/fundo.jpg') - Define a imagem de fundo. É necessário que a imagem esteja na pasta imagens e com o nome fundo.jpg.
background-size: cover - Garante que a imagem cubra toda a área do header, mesmo que seja necessário cortar partes da imagem.
background-position: center - Centraliza a imagem de fundo.
background-repeat: no-repeat - Impede a repetição da imagem.
display: flex; align-items: center; justify-content: center - Utiliza Flexbox para centralizar vertical e horizontalmente o conteúdo dentro do header.

header.intro .overlay { ... } - Seleciona a div com a classe overlay dentro do header.
position: absolute;: - Posiciona a camada de forma absoluta em relação ao container header.intro (que tem position: relative).
top: 0; left: 0; width: - 100%; height: 100%;: Faz com que o overlay cubra toda a área do header.
background: rgba(0, 0, 0, 0.4);: - Aplica uma cor preta com 40% de opacidade para escurecer a imagem de fundo.
z-index: 1;: - Define a ordem de empilhamento, colocando o overlay atrás do conteúdo (que tem um z-index maior).

header.intro .conteudo { ... } - Estiliza a div que contém o texto (dentro do header).
position: relative; e z-index: 2 - Garante que o conteúdo fique sobre o overlay, ficando visível.
text-align: center - Centraliza o texto.
color: #fff - Define a cor branca para o texto, melhorando a visibilidade sobre o fundo escuro.
padding: 20px - Adiciona espaçamento interno.

header.intro h1 { ... }: Estiliza especificamente o título da introdução.
font-size: 3rem;: Define um tamanho de fonte grande para o título.
margin-bottom: 20px;: Adiciona espaço abaixo do título.

section.sobre { ... }: Aplica estilos à seção “Sobre Mim”.
padding: 50px 20px;: Adiciona espaçamento interno para dar respiro ao conteúdo.
text-align: center;: Centraliza o conteúdo da seção.

footer { ... }: Estiliza o rodapé da página.
background: #333;: Define um fundo escuro.
color: #fff;: Define a cor do texto como branca.
text-align: center;: Centraliza o conteúdo do footer.
padding: 15px;: Adiciona espaçamento interno para o rodapé.
